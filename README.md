### E2E WEB UI Cypress Practise

Practice UI Automation with Cypress.

--------------------------------------------------------------

Terminal 1:

npm install --force

npm start

open your browser on http://localhost:4200/

--------------------------------------------------------------

Terminal 2:

npx cypress open
